# Open Cinema MCP

## Overview

A single MCP server that aggregates showtime data from 50+ London independent cinemas across 14 different booking platforms. The culmination of individual cinema MCPs into one unified tool for AI assistants.

## Planned Features

- Unified search across all London independent cinemas
- Filter by venue, date, time, genre, and accessibility
- Real-time data from Peckhamplex, Everyman, Curzon, Picturehouse, Rio, BFI, and more
- Built on the Open Cinema SQLite database

## Status

Coming soon — individual cinema MCPs are being built first, then unified into this single server.
