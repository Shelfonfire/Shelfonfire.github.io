# WhereToWork

## Overview

A curated, map-based directory of laptop-friendly workspaces across London. Find cafes, libraries, hotel lobbies and coworking spots with details on wifi, plugs, noise levels and more.

## Key Features

- Interactive map with 100+ London venues
- Filter by category, wifi, plug availability
- Detailed workspace info: coffee prices, stay limits, seating, noise level
- Opening hours, social links, review summaries

## Tech Stack

Next.js frontend with Leaflet/OpenStreetMap, FastAPI backend, Supabase PostgreSQL with PostGIS. Deployed on Vercel + Railway.
